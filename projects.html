RISC Processor

  • Designed a 16 bit 8 register multi cycle microprocessor to perform a given set of instructions.
  • Modelled architecture using Finite State Machine optimized for fast performance by executing tasks common to various instructions in parallel and hence reducing number of states.
  • Designed the datapath consisting of various entities such as ALU, Register File, RAM such that number of hardware components and clock cycles per instruction are minimized.
